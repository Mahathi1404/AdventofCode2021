# Advent Of Code 2021

### This repository contains my solutions for programming challenges of [Advent of Code](https://adventofcode.com/2021)

Advent of Code (AoC) is an advent calendar for programmers created by **Eric Wastl** and a small team, where each day of December you are provided with a new problem to solve with a language of your choice. Each year has a fantastic plot and story line you can follow along with as you solve each days challenge.
